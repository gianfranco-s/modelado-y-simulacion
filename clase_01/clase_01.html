2 ¿Qué es un modelo?

Es la representación simplificada de un fenómeno real. Sirve para entender el comportamiento de un sistema, anticipar su comportamiento futuro, o mejorar sistemas existentes.

La simulación permite estudiar el comportamiento del sistema sin necesidad de construirlo físicamente, o experimentar de forma riesgosa o costosa.

En esta materia nos orientaremos a los modelos matemáticos, que representan fenómenos mediante ecuaciones.

3 Clasificación de modelos

3.1 Según su comportamiento

  • Estáticos vs. Dinámicos: Si el tiempo es una variable relevante.
  • Lineales vs. No Lineales: Relación entre entrada y salida.
  • Tiempo Continuo vs. Tiempo Discreto: Cómo evolucionan las variables en el tiempo.
  • Determinísticos vs. Estocásticos: Si hay incertidumbre o aleatoriedad.

3.2 Según su derivación

  • Basados en leyes fundamentales (física, química).
  • De caja negra (empíricos): Basados en datos de entrada/salida, sin conocimiento interno.
  • De caja gris: Combinación de ambos.

Durante las clases 1 y 2, vamos a repasar e introducir temas de modelado y simulación.

4.1 Modelado de un tanque de agua

Consideremos el caso del llenado de un tanque de agua.

Nos interesa conocer cómo evoluciona el nivel de líquido dentro del tanque. Hay una cantidad de materia que ingresa al tanque y una cantidad de materia que egresa. En la relación de ambas, podremos hallar la evolución del nivel de fluido.

Magnitudes del ejemplo
Magnitud Símbolo Unidades SI
Tiempo t s
Caudal de ingreso q_i m^3/s
Caudal de egreso q_s m^3/s
Superficie del tanque C m^2
Volumen dentro del tanque V m^3
Nivel de fluido h m

El volumen dentro del tanque se calcula así: V = Ch

La ley de conservación de la masa se puede escribir así: \frac{dV}{dt} = q_i - q_s

Por lo tanto, reemplazando en la ecuación inicial: C\frac{dh}{dt} = q_i - q_s

El caudal de salida se puede considerar como proporcional a la presión en el fondo del recipiente (OJO). A su vez, la presión en el fondo del recipiente puede considerarse proporcional al nivel de fluido (OJO de nuevo): v_s \propto P_1 \propto h

Si la relación es de proporcionalidad, se puede plantear un valor de k que la convierta en igualdad: v_s \approx kh

Reemplazando en la ecuación de diferencia de caudales: C\frac{dh}{dt} = q_i - kh \dot{h}(t) = -\frac{k}{C} h(t) + \frac{1}{C} q_i(t)

Hemos arribado a una expresión que relaciona el nivel de fluido (variable bajo estudio), con el caudal ingresante (variable externa), y los parámetros del sistema. Supongamos que sabemos que el caudal de entrada, en el instante t_0=0 pasa de 0 a U_0. Para t>=0, la ecuación diferencial puede expresarse así: \dot{h}(t) = -\frac{k}{C} y(t) + \frac{U_0}{C}

4.3 Respuesta temporal analítica

Implica hallar una solución a la ecuación diferencial. Recordando las clases de cálculo, vemos que se trata de una:

  • Ecuación Diferencial Ordinaria – ODE, en inglés
  • De primer orden – hay una única derivada
  • Lineal – la variable no es argumento de ninguna otra función
  • No homogénea – hay un término independiente

La solución de esta ecuación diferencial es: h(t) = \frac{U_0}{k} + \left(h_0 - \frac{U_0}{k}\right) e^{-(k/C) t}

Consideremos el llenado desde vacío, con h_0 = 0 h(t) = \left( 1 - e^{-(k/C) t} \right) \frac{U_0}{k}

4.4 Gráfica de esta ecuación

Vamos a hacer un cambio de variables, para hacer más genérica la respuesta.

  • Constante de tiempo: C/k = \tau
  • Normalización: \hat{h} = h/(U_0/k)

Haciendo esos cambios: \hat{h}(t) = 1-e^{-t/\tau}

4.5 ¿Qué cosas estamos asumiendo?

OJO: v_s \propto P_1 \propto h Hay una relación entre las variables de presión, rapidez de salida y nivel, pero no es lineal, sino que está establecida por la ley de Bernoulli: P_h + \frac{1}{2} \rho v_h^2 + \rho g h = P_2 + \frac{1}{2} \rho v_s^2 + \rho g h_s

El lado izquierdo de la ecuación corresponde a la superficie del fluido. Allí, la presión es la presión atmosférica, dado que el recipiente se encuentra abierto: P_h=p_{atm}

El lado derecho corresponde al punto de salida, que también está abierto, de manera que P_h=p_{atm}. Si ponemos en este punto el origen de coordenadas, también podemos establecer que h_s=0 (OJO! No es válido si el tamaño del desagote es grande.)

Nótese que la presión del lado izquierdo de la ecuación no es la del fondo del tanque.

Tomando en cuenta esas consideraciones, podemos reescribir la ecuación de Bernoulli: 2gh = v_s^2 - v_h^2

Utilizando la definición de caudal, y reconociendo que entre el punto donde se mide h y el punto de desagote no se crea ni se destruye materia (OJO! No vale para flujo turbulento ni fluidos compresibles): q_s = q_h \longrightarrow v_s C_d = v_h C \longrightarrow v_h = \frac{C_d}{C} v_s 2gh = v_s^2 - \left( \frac{C_d}{C} v_s \right)^2 \longrightarrow 2gh = 1 - \left( \frac{C_d^2}{C^2} \right) v_s^2 q_s = \sqrt{\frac{2gC_d^2}{1-\frac{C_d^2}{C^2}}} \sqrt{h}

Donde C_d es la sección del desagote.

Este último caso, la ecuación descriptiva del sistema sería C\frac{dh}{dt} = q_i - q_s \longrightarrow C\frac{dh}{dt} = q_i - C' \sqrt{h}

Esta ecuación es claramente no-lineal, y la usaremos para el trabajo práctico.

5 Simulación del modelo lineal

Para este primer ejemplo, estableceremos constantes “a dedo”, para validar que el modelo sea coherente con nuestra expectativa de comportamiento.

Archivo “mys_clase_01a_modelo_lineal.m”

------------------------------------------------------------
% Modelos y Simulación
% Tema: modelo lineal de nivel
% Autor: Gianfranco Salomone
 
clc;
clear all;
 
% U0:      acción externa  [m^3/s]
% tfinal:  tiempo final    [s]
% ho:      nivel inicial   [m]

% k:       cte de prop     [m^2/s] (se deduce!)
% C:       sección tanque  [m^2]

% Inicialización de variables:
U0 = 10;
tfinal = 50;
h0 = 4;

% Constantes del sistema:
k = 0.3;
C = 1;
 
% Se define la ecuación diferencial
% primero la var indep, luego la dependiente
dh = @(t,h) -k*h/C + U0/C  % Es una función paramétrica
 
% Es posible evaluar la función paramétrica
% para t = 3, h = 4
dh(3,4)

% Array de puntos que cumplen con la condición de la función paramétrica
tspan = [0, tfinal]
[t, h] = ode23(dh, tspan, h0);

% Gráfica de la función y, una vez resuelta la ecuación diferencial:
plot(t, h);
title('Respuesta de h(t)');
xlabel('Tiempo (s)');
ylabel('Nivel del tanque (m)');
 
grid;

------------------------------------------------------------

5.1 Funciones

Ejemplo de función Para construir una función se usa la sintaxis:

function [ output_args ] = <nombre> ( input_args )
  <operaciones>
end

Se recomienda que el nombre de la función coincida con el nombre del archivo .m; el nombre de este último es el que usará MatLab para llamar a la función.

% Modelos y Simulación
% Tema: utilización de funciones -- main
% Autor: Gianfranco Salomone
 
clc;
clear all;

t_start = 0;
t_end = 20;
t_step = 0.1;

% Llamado a la función
[t, y, dy] = mys_clase_01b_func(t_start, t_end, t_step);

figure;

subplot(2, 1, 1);
plot(t, y, 'b-', 'LineWidth', 1.5);
xlabel('Time (t)');
ylabel('y(t)');
title('y = t + sin(t)');
grid on;

subplot(2, 1, 2);
plot(t, dy, 'r-', 'LineWidth', 1.5);
xlabel('Time (t)');
ylabel('dy/dt');
title('dy/dt = 1 + cos(t)');
grid on;

% Modelos y Simulación
% Tema: utilización de funciones -- func
% Autor: Gianfranco Salomone

function [z, y, dy] = mys_clase_01b_func(t_start, t_end, t_step)

  z = t_start : t_step : t_end;
  y = z + sin(z);
  dy = gradient(y, t_step);
 
end
